# Linux file system


---

## Common Directories

| Directory | Purpose                          |
|----------|----------------------------------|
| `/`      | Root of the file system         |
| `/home`  | User home directories           |
| `/etc`   | Configuration files             |
| `/var`   | Variable data (logs, spool, etc.) |
| `/bin`   | Essential user commands         |
| `/usr`   | User programs and data          |
